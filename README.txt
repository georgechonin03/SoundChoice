README

  SoundChoice е онлайн платформа за споделяне на аудио материали, 
 целяща да създаде пространство за потребители, които се нуждаят
 от специфично звучене.

  Препоръчително е използването на Visual Studio 2022 при отваряне
 и редактиране на приложението.

  Сайтът е достъпен на адрес http://soundchoice-001-site1.htempurl.com
